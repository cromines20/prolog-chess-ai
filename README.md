# Prolog Chess AI

This project is a rule-based chess AI implemented in Prolog.

## Project Type

Classical Rule-Based AI

## Project Overview

The goal of this project is to create a simple chess AI using Prolog. The program represents a chess board, generates legal moves, checks whether a king is in check, evaluates board positions, and chooses a move using minimax with alpha-beta pruning.

## Why Prolog?

Prolog was selected because it is well suited for symbolic reasoning, rule-based systems, and classical artificial intelligence. Chess is a good fit for Prolog because the game is based on formal rules, legal move constraints, and logical relationships between pieces and board positions.

## Main Features

- Board representation using Prolog facts
- Legal move generation
- Check detection
- Move application
- Heuristic board evaluation
- Minimax search with alpha-beta pruning
